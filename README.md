# Installation

0.  Install [Atom](https://atom.io/)
1.  Launch Atom
2.  Open **Preferences** > **Install**
3.  Search for `early-riser-syntax`
4.  Click **Install** to install it
5.  Preferences > Themes > Syntax Theme > **Early Riser**
